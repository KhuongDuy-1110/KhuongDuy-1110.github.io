3CSS Structure

This template includes 6 css files. Let's take a look closely at our stylesheet files:

  • style.css - common css styles for this template.
  • animate.css- a simple CSS library that saves you from writing a few more lines of CSS to animate elements on your website.
  • fonts.css - base styles for the Fontastic icon fonts.
  • bxslider.css - base styles for the bxslider.
  • owl.carousel.css and owl.theme.css - contain css styles for the carousel slider.

4JavaScripts

This template imports Javascript files. jQuery is a Javascript library that greatly reduces the amount of code that you must write. Let's take a look closely at our JavaScript files used with this template:

  • jquery-1.11.1.min.js - jQuery is a fast, small, and feature-rich JavaScript library.
  • jquery-migrate-1.2.1.js - jQuery Migrate plugin it was created to simplify the transition from older versions of jQuery.
  • jquery.scrollTo-min - a very light and highly customizable jQuery plugin, to scroll overflowed elements or the window itself.
  • jquery.sticky - sticky is a jQuery plugin that gives you the ability to make any element on your page always stay visible.
  • jquery.nav.js - Smooth scrolling and smart navigation when user scrolls on one-page sites.
  • modernizr-2.6.1.min.js - Modernizr is a JavaScript library that detects HTML5 and CSS3 features in the user's browser.
  • respond.min - Respond - A fast & lightweight polyfill for min/max-width CSS3 Media Queries (for IE 6-8, and more)
  • jquery-easing-1.3 - A jQuery plugin from GSGD to give advanced easing options.
  • owl.carousel - Touch enabled jQuery plugin that lets you create beautiful responsive carousel slider.
  • jquery.inview - a jQuery plugin that adds a bindable 'inview' event for detecting when an element is scrolled into view.
